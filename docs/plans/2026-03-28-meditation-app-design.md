# 静心 - 冥想引导应用设计

## 项目概述

一款极简风格的移动端冥想引导应用，帮助用户通过呼吸、正念、引导和放松等多种模式进行冥想练习。

## 核心功能

### 1. 冥想时长
- 用户可自定义冥想时长（1-60分钟）
- 默认提供快捷选项：5分钟、10分钟、15分钟、20分钟

### 2. 冥想模式（混合模式）
- **呼吸模式**：4-7-8呼吸法（吸气4秒、屏息7秒、呼气8秒），或自定义节奏
- **正念模式**：身体扫描引导，5个身体部位依次放松
- **引导模式**：冥想导师语音/文字引导（可扩展）
- **放松模式**：可视化场景（海滩、森林、雨夜、星空）

### 3. 冥想流程
- 开场：柔和铃声 + 3秒倒计时
- 主体：根据选择模式进行
- 结束：渐出铃声 + 统计展示（时长、心跳平静度）

## 技术架构

### 框架
- Flutter 3.x（移动端优先）
- 状态管理：Riverpod 或 Provider

### 核心组件

#### HomeScreen（首页）
- 欢迎语 + 今日心情选择
- 时长选择器（滑块 + 快捷按钮）
- 模式选择卡（4个模式图标）
- 开始冥想按钮

#### MeditationScreen（冥想页面）
- 圆形进度指示器（动画）
- 呼吸动画圆环（放大缩小）
- 引导文字显示区
- 场景可视化背景（放松模式）
- 暂停/继续按钮
- 提前结束按钮

#### HistoryScreen（历史记录）
- 累计冥想时长
- 连续打卡天数
- 每次冥想记录（日期、时长、模式）

#### SettingsScreen（设置）
- 默认冥想时长
- 铃声开关/音量
- 引导语语言
- 深色/浅色主题切换

## UI/UX 设计

### 配色方案（极简静谧）
- 主背景：#0D1117（深空黑）
- 次背景：#161B22（深灰）
- 主强调：#58A6FF（宁静蓝）
- 次强调：#8B949E（柔和灰）
- 文字主色：#F0F6FC（纯白）
- 文字次色：#8B949E（灰色）

### 动画效果
- 呼吸圆环：缩放动画（ease-in-out），2-8秒周期
- 进度条：平滑过渡
- 页面切换：淡入淡出
- 按钮反馈：轻微缩放 + 波纹

### 字体
- 标题：Bold，24-32sp
- 正文：Regular，16sp
- 引导文字：Light，18-20sp
